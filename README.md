# 🇿🇲 Zambia Quiz - Android App

An interactive educational quiz application about Zambia, featuring 500+ unique questions across 8 categories.

## Features

- ✅ **500+ Unique Questions** - Comprehensive coverage of Zambian topics
- ✅ **8 Categories** - History, Geography, Culture, Wildlife, Economy, Sports, Politics, Famous People
- ✅ **3 Difficulty Levels** - Easy, Medium, Hard
- ✅ **Timed Quizzes** - 10-second timer per question
- ✅ **Multiple Quiz Lengths** - Choose 10 or 20 questions
- ✅ **Sound Effects** - Audio feedback for correct/wrong answers
- ✅ **Haptic Feedback** - Vibration on answer selection
- ✅ **Beautiful UI** - Zambian-themed design with flag colors
- ✅ **Background Slideshow** - Scenic images of Zambia
- ✅ **Score Sharing** - Share your results with friends
- ✅ **Progress Tracking** - See your performance stats




## Technologies Used

- **Language:** Kotlin
- **Libraries:**
  - ViewPager2 - Image slideshow
  - Glide - Image loading
  - Gson - JSON parsing
  - Material Design Components
  - RecyclerView - Category grid

## Installation

### Prerequisites
- Android Studio Hedgehog or later
- Android SDK 24+
- JDK 17+

### Steps
1. Clone the repository
```bash
git clone https://github.com/Emmanuel-cpp/Zambia-Quiz-Mobile-App.git
```

2. Open project in Android Studio
3. Sync Gradle files
4. Run on device or emulator

## How to Use

1. **Launch App** - Splash screen displays
2. **Main Menu** - Choose "Start Quiz" or "Quick Play"
3. **Select Category** - Pick difficulty and category
4. **Choose Quiz Length** - 10 or 20 questions
5. **Answer Questions** - 10-second timer per question
6. **View Results** - See your score and stats
7. **Share** - Share your achievements!
